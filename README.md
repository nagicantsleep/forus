# Forus Study Focus Web App

A lightweight web application built with React, TypeScript, and Vite that helps you stay focused while studying.

## Features
- Pomodoro‑style timer with customizable sessions
- Simple task tracking without requiring a login (session‑based)
- Background sounds and music options, including YouTube integration
- Two themes: **Minimal** for distraction‑free work and **Incredible** for a richer visual experience
- Responsive design with modern UI aesthetics

## Getting Started
```bash
npm install
npm run dev   # Starts the development server
```
